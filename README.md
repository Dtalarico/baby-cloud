# Baby Cloud

Personal Cloud infrastructure lab built on RHEL demonstrating:

- Linux system administration
- Networking fundamentals
- Container workloads
- Infrastructure automation

## Project structure

- docs/ - Architecture and infrastructure documentation
- diagrams/ - Infrastructure diagrams
- scripts/ - Automation scripts for environment setup

## Current Components

- RHEL virtual machine host
- Internal lab network
- Container runtime environment
- Infrastructure automation scripts 

## Future Expansion

- Reverse proxy
- Monitoring stack
- Multiple VM nodes
- Load balancing
